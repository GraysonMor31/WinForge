# WinForge 🔨

> An open-source way to make Windows clean, optimized, bloat-free, and ad-free.

---

## Overview

WinForge is a lightweight, open-source tool for stripping Windows down to what matters. Whether you're setting up a fresh install or cleaning up an existing machine, WinForge automates the tedious work of debloating, optimizing, and hardening your Windows environment.

---

## Features

- 🧹 **Debloat** — Remove pre-installed bloatware and unnecessary built-in apps
- ⚡ **Optimize** — Tune system settings for performance and responsiveness
- 🔒 **Harden** — Apply sensible privacy and security defaults
- 🎛️ **Modular** — Run only the modules you need
- 💻 **Open Source** — Fully transparent, community-driven

---

## Getting Started

```powershell
# Clone the repository
git clone https://github.com/GraysonMor31/WinForge.git
cd WinForge

# Run WinForge (as Administrator)
./winforge.ps1
```

> ⚠️ Run as Administrator for full functionality.
